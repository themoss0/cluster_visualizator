## О проекте 
Данный проект расчитан на практику в отработке задания 27 ЕГЭ по информатике 2026 года. 

## Установка
- Скачайте и разархивируйте исходный код.
- Установите библиотеки: `pip install matplotlib numpy scipy`


# Запуск
- Разместите ваш файл (соответствующий формату X Y без разделителей) в корневую папку `cluster_visualizator`
- Запустите, введите название файла.
